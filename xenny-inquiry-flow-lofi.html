Xenny · 문의(상담) MVP

사용자 앱 — 문의 플로우 (Lo-fi)

로그인 사용자가 문의를 등록하고 답변을 확인하기까지의 흐름입니다. 구조·전환·규칙만 담은 와이어프레임이며, 비주얼 디자인은 별도로 진행합니다. 범위는 사용자 앱에 한정합니다.

화면 정상 전환 되돌아가기 · 루프 외부 진입 (알림) 활성 버튼
01

내 문의 (빈 상태)

본인 문의만 조회. 첫 문의 작성을 유도. 비로그인은 기존 채널톡 사용(범위 외).

02

문의 작성

제목 최대 200자 · 내용 최대 2000자. 카테고리·첨부파일은 입력받지 않음.

03

작성 · 글자수 초과

제목 200자 또는 내용 2000자 초과 시 인라인 에러 + 등록 비활성. (인수 조건)

04

등록 완료

문의는 OPEN(답변 대기)으로 생성. 토스트 노출 후 목록/상세로 이동.

05

내 문의 (목록)

표시 항목: 제목 · 상태 · 등록일 · 답변일 · 답변 미리보기.

06

상세 · 답변 대기

문의 내용만 표시, 입력창 없음(단건 처리). 상태 OPEN.

07

상세 · 답변 완료

운영자 답변 + 답변일 표시. 운영자 이름 미노출('운영자 답변'). 재답변 불가 → '추가 문의'는 신규 문의로.

알림

답변 도착

운영자 답변 등록 시 notification_queue에 적재 → 도착 알림. 탭하면 해당 상세로 진입.
